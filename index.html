Méthodologie et sources

Les calculs d'émissions sont basés sur les données issues de :

  • Consommation énergétique : RTE (mix électrique français 2022)
  • Facteurs matériaux : Base INIES (2022)
  • Transport : Données Ademe - Carbone4 (2022)

Les facteurs d’émission sont exprimés en kg eq CO₂ par unité fonctionnelle (kWh, tonne, km). Ces données peuvent être modifiées ou mises à jour par les administrateurs du projet via les fichiers CSV.
